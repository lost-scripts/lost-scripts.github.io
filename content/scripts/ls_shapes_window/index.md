## Main Features

- **Shape Management**: Easily create, delete, and modify shapes.
- **Intuitive UI**: User-friendly interface for smooth operations.
- **Advanced Tools**: Access to a variety of advanced shape manipulation tools.

<br>

## Installation (Two Ways)

| Manual                        | Automatic                     |
| ----------------------------- | ----------------------------- |
| 1. Unzip the downloaded file. | 1. Unzip the downloaded file. |
| 2. You'll get one or more of the following folders: `Menu / ScriptResources / Tool / Utility`.           | 2. From Moho's main menus, go to "Scripts -> Install Script..." and a window with some info will appear.           | 
| 3. Drag & Drop (or Copy & Paste) them all into the Scripts folder of your [Custom Content Folder][3-1a]. | 3. Press "Select A Script Folder" button, browse to the just uzipped folder (e.g. ls_shapes_window) and select it. |
| 4. Restart Moho® or press "Alt + Shift + Ctrl + L" to Reload Tools And Brushes and that should be all.   | 4. That should be all. See chapter [23.17 Install Script...][3-2a] of Moho® user's manual for further details.     |

> ⚠ WARNING: Please, make sure you have uninstalled every Lost Script on your system before removing any of these shared resources or they may start throwing errors or stop working. For uninstalling a script, just remove any file and folder matching its name and restart Moho® or Reload Tools And Brushes if necessary.

<br>

## Usage

1. **Opening**: Open the LS Shapes Window from the Moho® scripts menu.
2. **Shape Operations**: Use the interface to perform various shape operations.
3. **Shortcuts**: Utilize keyboard shortcuts for faster operations.
