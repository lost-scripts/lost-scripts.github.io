# LS Shapes Window

Herramienta versátil que te ayuda a gestionar y manipular formas en Moho® con facilidad. Este script proporciona una interfaz fácil de usar para realizar operaciones complejas sin esfuerzo.

## Features

- **Shape Management**: Easily create, delete, and modify shapes.
- **Intuitive UI**: User-friendly interface for smooth operations.
- **Advanced Tools**: Access to a variety of advanced shape manipulation tools.

## Installation

To install LS Shapes Window, follow these steps:

1. **Download**: Download the latest version from [GitHub](https://github.com/lost-scripts/ls-shapes-window).
2. **Extract**: Extract the files to your Moho® scripts directory.
3. **Run**: Restart Moho® and find LS Shapes Window in the scripts menu.

## Usage

1. **Opening**: Open the LS Shapes Window from the Moho® scripts menu.
2. **Shape Operations**: Use the interface to perform various shape operations.
3. **Shortcuts**: Utilize keyboard shortcuts for faster operations.
